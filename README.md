# Homebrew Cask for SlashTable

## Install

```bash
brew tap slashtable/cask
brew install --cask slashtable
```

## Update

```bash
brew update
brew upgrade --cask slashtable
```

SlashTable also has a built-in auto-updater.

## Uninstall

```bash
brew uninstall --cask slashtable

# To also remove all data and preferences:
brew zap slashtable
```
